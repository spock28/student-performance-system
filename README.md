# Student Performance Prediction System

## Objective
This project predicts student performance using a neural network.

## Inputs
- Attendance
- Assignment Marks
- Test Marks

## Output
- Performance Level (Poor, Average, Good)

## Methodology
A neural network (MLPClassifier) is trained using sample student data.

## Model Details
- Algorithm: MLPClassifier
- Hidden Layer: 5 neurons
- Library: scikit-learn

## Result
The model predicts student performance based on input features.

## Example Output
Predicted Performance: Average
